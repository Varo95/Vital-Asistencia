On this screen we can see the registered users, the amount that we have and a button who help us to insert new user. Once we add a new one, the counter on the top will increase, if we delete a user, the counter will decrease.

If we drag a element on the list from right to left, it will show us two buttons.

The red button deletes the user. If we press it the app will delete it directly, showing us a text with the affiliatte number.
