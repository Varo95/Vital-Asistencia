On this screen we will put the user data.
The fields are:

  • 1-(Optional)Photo
  • 2-(Optional)Day of the week (If you press the plus button you can add a new one that is not in the list)
  • 3-Date: It will be the assigned date of the task
  • 4-(Optional)Prepare food: If he is active it means that we would have to make him food
  • 5-Email: The user email
  • 6-Address: The address of the user we need to go to
  • 7-Affiliate Number: It is a number provided by the company to identify the user


    • Insert


    If we have pressed the add button on the list screen, the empty form will load us.
    If we insert a value in a field that does not meet the conditions, it will notify us with an error and show a brief help.
    If we leave any mandatory field empty and press the save button, the empty field (s) that is / are mandatory will be marked in red.
    The delete button will be disabled, because if we want to cancel its creation, just press the back button.


    Update


    If we have pressed the update button or on a user in the listed activity, this activity will load us with the data of said user already filled in.
    If we modify any value in a field and it does not meet the conditions, it will notify us with an error and showing a brief help.
    If we leave any mandatory field empty and press the save button, the empty field (s) that is / are mandatory will be marked in red.
    By default the affiliate number field cannot be modified. In this case, a new user should be created with the same data.
    The delete button is available when a user is updated, if we click on it a pop-up window will appear to confirm its deletion.
    If we do the latter, it will return us to the list screen.
